Selection Process (Brief)

Common Steps (for all jobs):
1. Online Application
2. Written Exam (CBT / OMR)
3. Physical Test (mandatory in Defence & Police)
4. Medical Examination / Document Verification
5. Final Merit List

For Defence Jobs (Air Force / Army / Navy / NDA):
• Written Exam (Physics, Maths, English, Reasoning, GK)
• Physical Fitness Test (1.6 km run, Push-ups, Sit-ups, Beam etc.)
• Adaptability Test / SSB Interview (for NDA)
• Strict Medical Test
For Non-Defence Jobs (SSC, Railway, Police etc.):
• Tier-1 / CBT Written Exam (Reasoning, Quant, English, GK/Current Affairs)
• Skill Test / Typing Test (in some posts)
• Document Verification & Medical
